# react-nipd-types

## Install

```bash
npm install --save react-nipd-types
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'react-nipd-types'
import 'react-nipd-types/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
